Equilibres des produits


Les équilibres des produits agricoles et alimentaires sont montrés sous une forme standardisée. La standardisation présente ces données sous une forme moins détaillée pour un nombre restreint de produits sans causer de perte significative des variables de base contrôlant le secteur agricole. Les produits choisis incluent les équivalents de leurs produits dérivés tombant dans le même groupe de marchandises, mais excluent les équivalents de leurs sous-produits et de leurs produits dérivés qui, par transformation, changent de nature et deviennent partie de groupes de produits différents. Un certain nombre d'agrégats de produits ont été inclus pour offrir des informations synthétiques. Certains d'entre eux sont inclus dans le but de simplifier l'extraction de toutes les produits composants. Les données montrées dans les agrégats de produits représentent la somme des produits composants comme présenté dans ce domaine (forme standardisée).

Couverture

La liste de produits dans ce domaine a été généralement limitée aux produits primaires - à part le sucre, les huiles et les graisses et les boissons. Autant que possible, le commerce des produits transformés est exprimé en équivalent primaire dans le produit dont ils dérivent. Le riz est exprimé dans l'équivalent blanchi.
